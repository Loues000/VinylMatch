About VinylMatch

Connect digital playlists with physical releases

What is VinylMatch?

VinylMatch pairs Spotify tracks with Discogs releases, making it easier to find the right pressing for a playlist. Paste a Spotify playlist link and VinylMatch generates Discogs links and vendor search links per track.

Key Features

  • Spotify integration — OAuth login with session cookies.
  • Smart matching — Progressive Discogs search with safe fallbacks.
  • Vendor links — HHV, JPC, Amazon and custom vendors via configuration.
  • Recent playlists — Quickly jump back to your last searches.
  • Curation tools — Manual matching for edge cases.
